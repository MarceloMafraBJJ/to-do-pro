# TODO PRO

![todo](https://github.com/MarceloMafraBJJ/to-do-pro/assets/84472778/111bda35-6dd6-4825-be19-2302fa009dbd)



Features:

- Nextjs
- TailwindCSS
- Clerk
- shadcn/ui

### Install packages

```shell
npm i
```

### Setup .env file

```js
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
DATABASE_URL=
```

### Run

```shell
npm run dev
```
